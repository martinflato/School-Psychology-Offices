This repository contains information on school psychology offices established in Norway between 1947-1983 that is used in Flatø et al. (2026). The file spos_by_observations.dta contains one observation per office per year, and includes information on the number of staff by professional background. The file spos_by_municipality.dta contains one observation per municipality, and the year in which the office was established and staffed. The file sources.dta provides an oveview of the sources for the data. The data was created by Martin Flatø.
